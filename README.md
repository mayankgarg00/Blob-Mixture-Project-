🧪 Blob Mixture Project

A hands-on implementation of Gaussian Mixture Models (GMM) to explore probabilistic clustering on synthetic “blob” datasets.

📌 Overview

Clustering is often treated as a hard assignment problem (like K-Means), but real-world data is rarely that simple.

This project demonstrates how mixture models provide a more flexible, probabilistic approach by modeling data as a combination of multiple distributions. Using blob datasets, we explore how Gaussian Mixture Models (GMM) can uncover hidden structures in data.

🚀 Key Highlights
* 🔍 Implements probabilistic clustering using GMM
* 📊 Visual comparison between actual and predicted clusters
* 🧠 Uses Expectation-Maximization (EM Algorithm)
* 🎯 Works on synthetic blob datasets for clear intuition
