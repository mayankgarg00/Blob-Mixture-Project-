Blob Mixture Project
📌 Overview

This project focuses on implementing a mixture model (such as a Gaussian Mixture Model) on synthetic or real “blob” datasets to perform clustering and analysis.

It demonstrates how multiple distributions can be combined to model complex data.

🎯 Objectives
Generate or use blob-like datasets
Apply mixture models (e.g., Gaussian Mixture Model)
Visualize clustering results
Understand probabilistic clustering techniques
🧠 Concepts Used
Clustering
Expectation-Maximization (EM Algorithm)
Probability Distributions
Gaussian Mixture Models (GMM)
🛠️ Tech Stack
Python
NumPy
Matplotlib / Seaborn
Scikit-learn
